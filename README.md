# s9noru - Кастом шифт сайт

Сайт в стиле iOS 26 с кастом шифт эффектами.

## Ссылки
- Telegram TG: [@fucknoru](https://t.me/fucknoru)
- Telegram BIO: [@s9norubio](https://t.me/s9norubio)

## Особенности
- Анимированный фон с сеткой и волнами
- Кастом шифт переключатель между TG и BIO
- 3D кнопки с эффектами
- Адаптивный дизайн
- Градиенты и эффект стекла (blur)

## Технологии
- HTML5
- CSS3 (анимации, градиенты, backdrop-filter)
- JavaScript
